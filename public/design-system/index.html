Usage Rules
Charcoal 900 → 표지 배경, 헤더, 섹션 구분선 1px (full-width)
Point Blue 600 → 강조 숫자, 딜 시그널, Accent Bar 3px
Blue 50/100 → 하이라이트 행 배경, 강조 카드 배경
Charcoal 100/50 → 기본 테이블 배경, 카드 표면

06 Do / Don't
반드시 지킬 것
✓표지 배경: Charcoal 900 (#1A1A1A)
✓강조 숫자: Point Blue 600 (#1D4ED8)
✓섹션 구분: 1px solid Charcoal 900 full-width
✓수치 데이터: Monospace 폰트 고정
✓Blue Accent Bar 3px — 섹션 상단 필수
✓페이지 사이드 마진: 20mm (PDF 기준)
✓border-radius 최대 12px
절대 금지
✕그라디언트 / 그림자 / Blur / Glow
✕포인트 컬러 3종 이상 동시 사용
✕border-radius 12px 초과
✕font-weight 700 / 800 이상
✕장식적 아이콘 · 이모지 남발
✕빨간색(#FF0000) 강조 사용
✕5가지 이상 컬러 동시 사용
